# Implementation Plan: Robust Network Connectivity

**Branch**: `005-robust-network-connectivity` | **Date**: 2026-03-24 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/005-robust-network-connectivity/spec.md`

## Summary

Replace the single-consumer `Channel<ViewCommand>` in MessageRouter with a per-subscriber broadcast pattern using `ConcurrentDictionary<Guid, Channel<ViewCommand>>`. Fix MCP SSE endpoint accessibility through DCP proxy by configuring `isProxied: false`. Consolidate all network debugging findings into NetworkProblems.md with container environment documentation. Rebase on main to inherit 004-camera-smooth-demos fixes (kill.sh .dll patterns, ConcurrentQueue drain, body-not-found hold).
